IDEA
Our team’s goal is to build a quality, user-friendly blog platform where:

Readers can discover and read well-organized, high-quality articles.
Content creators can effortlessly write, categorize, tag, and publish their articles.
Administrators can efficiently manage categories, moderate comments, and ensure the platform remains engaging and clean.
FEATURES (early)
1. Loggin in
Secure login process for platform access.
Verification of successful login before dashboard access.
2. Article Management
Write new articles, add tags, and organize content into categories.
Preview functionality ensures articles are ready before publishing.
Easy-to-use publish workflow:
Write → Categorize → Tag → Preview → Publish.
3. Content Organization
Manage categories:
Add new categories.
Edit existing ones (delete if neccesary).
Maintain clear article organization for better content discovery.
4. User Experience
Simple navigation to read articles.
Articles are structured and categorized for easy access.
5. Comment Management
Administrators can:
View comments on articles.
Approve or delete comments to ensure discussions remain constructive.
6. User Logout
A clean logout process ensures user security and session management.
